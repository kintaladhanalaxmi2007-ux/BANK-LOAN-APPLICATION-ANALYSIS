Project Overview

This project presents an interactive Bank Loan Application Analysis Dashboard built using Microsoft Excel. The dashboard provides deep insights into loan performance, borrower behavior, and approval trends.

It helps stakeholders understand financial data through visual storytelling, KPIs, and dynamic charts, enabling better decision-making in loan management.

Problem Statement

The objective of this project is to analyze bank loan applications and answer key business questions such as:

What is the overall loan performance?

How many loans are good vs bad?

Which regions have the highest loan demand?

What borrower profiles are most common?

The goal is to transform raw financial data into meaningful insights using Excel dashboards.

Key Insights

$183M+ received vs $169M funded, showing healthy recovery

Loan applications are increasing in recent months

California (CA) and Texas (TX) lead in total applications

Borrowers with 10+ years of experience take the most loans
